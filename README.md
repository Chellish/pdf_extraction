## 使い方

以下の手順に従って、このプロジェクトを使用してください。

1. `PDF抽出.exe` を実行します。
2. 実行後、抽出対象のPDFファイルと抽出条件が記載されたCSVを選択してください。
   - **CSVファイルの形式**: ファイル名、抽出開始ページ、抽出終了ページの3項目で構成され、**ヘッダなし、カンマ区切り**としてください。
3. 抽出開始ボタンを押下すると自動的に処理が走ります。
4. 抽出終了後、GUIに結果が表示されます。
